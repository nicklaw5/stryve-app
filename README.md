# Stryve
A self-hosted chat/messaging application, heavily inspired by [Discord][discord] and [Slack][slack].

## Installation

### Requirements

+ [nodeJs][nodejs]
+ [npm][npm]


### Install

```bash
$ git clone https://github.com/nicklaw5/stryve
$ cd stryve
$ npm i # Install needed modules
$ npm run w-app # Run the app
```
